# Система информационного поиска (Задания 2-4)

Данный проект представляет собой реализацию инвертированного индекса с поддержкой LSM-подобного хранения и фильтрации по метаданным (датам).

## Реализованный функционал

### Задание 2: Базовый индекс
- Roaring Bitmaps: Использование библиотеки pyroaring для эффективного хранения и пересечения списков ID документов.
- NLP пайплайн: Токенизация, очистка от стоп-слов и стемминг (Snowball) с использованием библиотеки nltk.
- LSM-структура: Реализована концепция MemTable (хранение в RAM) и автоматический сброс (flush) данных на диск при достижении лимита. Поиск осуществляется прозрачно по всем компонентам.

### Задание 3: Продвинутый поиск
- Префиксный поиск: Эффективный поиск по началу слова через обход словаря.
- Wildcard поиск: Реализован с использованием k-gram (триграмм). Запрос разбивается на подстроки, выполняется предварительная фильтрация по словарю с последующей проверкой регулярными выражениями.

### Задание 4: Временные метаданные
- К документам добавлены атрибуты: created_at, valid_from, valid_to.
- Реализована фильтрация по диапазонам дат.
- Поддержка сложных булевых формул (пересечение текстовых запросов с временными фильтрами через побитовые операции битмапов).
